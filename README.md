# gmaps-api-game 

COMP 484 - Project 5: Google Maps API - JS Game 

[GitHub Pages Link](https://mel-rossi.github.io/gmaps-api-game/)

[Google Maps Platform Docs - Demo Key](https://developers.google.com/maps/documentation/javascript/demo-key)

## Configure Google Maps API

1. Rename the file `config.template.js` to `config.js`
2. Replace `API_KEY_HERE` with your Google Maps API key
   - Demo Key can be obtained in the Link provided above
3. Open `index.html` in your web browser of choice
